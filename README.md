# NikitkoSBP
Система Безопасного Подключения позволяет сделать работу некоторых сервисов, ну типо, круче?
Ну в общем, это не реклама и ни фейк. Ютуб не работает, а смотреть хочеца. Сборник сервисов СБП (Система Быстрого Подключения) позволяет смотреть его в Ultra 4K Super Pro Max Mango TrollFace ну и так далее. Короче да.
